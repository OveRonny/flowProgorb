# frontend-vue

Vue frontend for FlowProgorb.

## Project setup

```sh
npm install
```

## Environment variables

Create a `.env` file for local development based on `.env.example`.

Required variables:

```sh
VITE_API_BASE_URL=http://localhost:3000
```

For production builds, set `VITE_API_BASE_URL` to your deployed backend URL.
See `.env.production.example` for the expected format.

## Development

```sh
npm run dev
```

## Production build

```sh
npm run build
```
